Plugin_development.Generic.generic module¶

Plugin_development.Generic.generic.main(data, results, **kwargs)[source]¶

Function extracts results from dicom file, using the private dicom tag configured in the config-xml [private_group,private_element] and writes the results to the results-object for further processing.

For an example, check:
Tools/Generic/create_dicom.py (creates a dicom file) Testing/Generic/generic_input.xml (configures where and what to extract from the dicom file)
