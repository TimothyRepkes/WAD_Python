Plugin_development.Bucky.Bucky_EdgeMTF.Bucky_EdgeMTF module¶

A very crude and simple implementation to estimate the MTF of an Edge phantom.

Plugin_development.Bucky.Bucky_EdgeMTF.Bucky_EdgeMTF.MTF_main(data, results, **kwargs)[source]¶

Function estimates MTF from an exposure of an edge phantom. Workflow: 1. From data object obtain first instance 2. Remove an arbitrarily chosen offset, which is hardcoded (20 pixels). This needs improvement. 3. Define what the minimal stepsize between adjacent pixels is 4. For each row (hardcoded range!, needs improvement) determine where the step is located 5. Sum the edge curves and finally take the Fourier transform to calculate the MTF
